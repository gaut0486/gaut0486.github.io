Key Learnings:

Dl tag:
The dl tag is used to indicate a special type of list, that allows you to display a list item, followed by the list item description.
Aside
The aside tag is used to indicate secondary content.
Blockquote
The Blockquote tag allows the developer to create a large standalone qoute from another source.
Citation
The Cite tag is used to cite another source, such as a persons name,a url or a book.
Mark
The Mark tag is used to highlist a piece of text for reference.
